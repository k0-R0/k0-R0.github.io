Address Book

A command-line contact manager in C with file-backed storage and simple search. Use the sections below for the long-form article, build notes, and a CLI cheatsheet alongside the repository link.

  • Repository
  • Config
  • Cheatsheet

Add, list, edit, and delete contacts; persist them between runs; search without leaving the terminal. The scope stayed small on purpose so the interesting parts stay in C, file I/O, and shaping the CLI.

Highlights I expect to keep in the final article:

  • Structuring records in memory and on disk so saves and loads stay predictable.
  • Guarding string handling, file paths, and cleanup when input goes wrong.
  • Clear prompts and errors instead of clever but opaque output.
